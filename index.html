Typeboost.css is a set of CSS rules to ensure readable, scalable and responsive typography. REMs are used for font sizes, paddings, and margins. Can be used as typography normalizer. No dependencies. Wrap your HTML content with class .col and you are done. You will also like 16pixels

		<div class="col">
